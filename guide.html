如何使用簡報助手 與 NotebookLM 製作簡報？

💡 核心工作流：素材原子化 -> 結構化 YAML -> NotebookLM 簡報生成

設定 API Key 與上傳素材

首先，您需要設定 Google Gemini API Key 才能啟用 AI 大腦。 👉 免費申請 Gemini API Key
接著，您可以直接輸入文字，或拖曳上傳您的教學文件（PDF, Word, Text）或圖片。
系統支援多組 Key 輪替使用，以確保生成穩定性。

⚠️ 檔案上傳限制與注意事項

  • PDF 大小限制 (15MB)：本系統採純前端架構，受限於 Google Gemini API 官方規定，單次請求的總大小不可超過 20MB。建議單一 PDF 檔案控制在 15MB 以內，避免產生 400 Bad Request 錯誤。
  • 頁數與 Token 建議：為了確保 AI 分析精準不漏看，建議單次上傳 20~30 頁以內 的單一主題講義。
  • 關於 MP4 影片支援：目前版本已移除對 MP4 等影片格式的支援。因大型影片必須透過具有後端伺服器的 File API 才能安全上傳，若直接在前端上傳會導致瀏覽器記憶體崩潰並發生無限卡頓。

生成簡報結構 (Construction Blueprint)

選擇您的目標對象（例如：國中生）與頁數，點擊「開始生成」。AI 會分析您的素材，並產出兩份文件：

  • 簡報大綱 (Markdown)：供您快速預覽內容。
  • 結構化資料 (YAML)：這是最重要的部分，包含了簡報的完整指令碼。

匯入 Google NotebookLM

這是 SlideAgent 最強大的功能。點擊結果區的 「下載 YAML 為 .txt」 按鈕，保存檔案。

📂 前往 Google NotebookLM 並建立新筆記本。
⬆️ 將剛剛下載的 .txt 檔案上傳作為來源 (Source)。
✨ 在 NotebookLM 中，指令 AI：「請根據來源檔案的 YAML 結構，為我生成這份簡報的投影片內容」。

🚀 20260302 更新說明

🎨 1. 使用者介面 (UI) 全面進化

  • 折疊式選單 (Accordion Layout)：這是一次重大的版面升級！原本左側長長一列的設定選項，現在被優雅地收納進兩個折疊區塊：
    • 📦 內容與架構：包含講者身分、目標受眾、目的架構、簡報頁數。
    • 🎨 視覺設計與產出：包含美術風格、YAML 匯出模式、自動加結語頁、AI 深度優化。
  • YAML 匯出模式強化與說明：明確定義了兩種產出模式的用途，讓使用者能依據後續需求選擇：
    • 📄 潔淨內容 (高自由度)：預設模式。此模式下產出的 YAML 只包含純粹的文字內容與邏輯架構。這賦予 NotebookLM 極高的擴寫自由度，它將能「隨機決定」各種元素的擺放與生成，適合需要 NBLM 自由發揮創意的場景。
    • 🎨 豐富指令 (多頁控制)：進階模式。這是一種「工程控制思維」的產出。在這個模式下，系統會將設定的「美術視覺風格」、「版面配置」、「視覺描述」等指令，強制寫死在每一頁的 YAML 中，猶如給 NotebookLM 下達精準的生圖與排版指令。適合需要嚴格控制每張簡報最終視覺呈現的專業需求。
  • 智慧型折疊連動：面板狀態會根據您的使用模式自動切換，提供最乾淨的視覺體驗（AI擴寫預設展開內容，現有大綱模式則預設展開視覺設定）。
  • 版面動線統一：將原本位於最上方標題列的「AI 設定（API Key輸入）」按鈕，移動至左側「參數設定」區塊的右上角。從輸入 Key 到調整參數，所有的操作現在都完美集中在左側了。
  • 放寬頁數限制：根據需求，「簡報總頁數」的上限設定，已經從原本的 20 頁大幅放寬至 60 頁。
  • 上傳提示文字更新：明確告知可支援圖片檔拖曳上傳。

🧠 2. AI 生成邏輯強化與修復

  • 「自動加結語頁」火力全開（支援現有大綱模式）：
    現在 AI 在精準轉換完您提供的所有大綱內容後，會強制在最後額外生成一頁「深度省思（deep_reflection）」 的結語頁。這不僅不會吃掉您的原始大綱，還能完美達成「外加一頁（N+1 頁）」的需求！
  • 多媒體上傳機制與最佳實踐說明：
    這版釐清了一個重要的 API 底層限制：當同時上傳「文字量極大的 PDF」加上「多張圖片」時，Gemini 語言模型的注意力（Attention Mechanism）高機率會被 PDF 的文字篇幅給吸走，導致「漏看」圖片內容。

    ✅ 最佳實踐建議：為了確保 AI 產出的準確性，強烈建議使用者「盡量不要混用不同型態的檔案」。如果您需要以圖片為核心（例如圖片敘事），請只上傳圖片；如果需要加上文字補充說明，建議您直接使用系統建議的方法——將 PDF 轉存成純文字大綱，或是直接將文字貼在「文字輸入框」中。 「純文字 + 圖片」的組合能讓底層的 Gemini 發揮100%的最佳效能，絕不漏拍！
  • 穩定性修復：消除了在現有大綱模式下會導致程式無法順利產出簡報的核心 ReferenceError 錯誤。

✨ 20260301 新功能
🎯 多重簡報目的架構： 現在您可以指定簡報的「骨架」。選擇「教學課程」會強制要求 AI 加入學習目標與隨堂測驗；選擇「專案報告」則會嚴格遵循痛點分析到行動呼籲的商業邏輯；而「故事演講 (TED)」則會捨棄死板的列點，改用情境帶入的敘事曲線。
🧠 AI 深度優化 (Self-Correction)： 在參數設定最下方開啟此開關後，AI 會在初稿完成時進行第二次「自我審查」，由內建的虛擬簡報教練重新潤飾文字、邏輯除錯與排版升級。雖然需較長等待時間，但能獲得極致洗鍊的大綱品質。
🎨 自訂大師級視覺風格 (附設定規範)： 選擇「✨ 自訂視覺風格」後，您可以徹底解放 AI 的詠唱潛力。為確保 NotebookLM 產圖驚艷，強烈建議您的指令包含以下 3~4 個維度：
  • 主體材質/媒材：例如「水彩畫 (Watercolor)」、「3D 渲染 (3D render)」、「復古拍立得底片」、「幾何向量插畫」。
  • 光影與氛圍：例如「溫柔的自然採光 (Natural soft lighting)」、「賽博龐克霓虹光」、「高對比戲劇性光影」、「夢幻且空靈的」。
  • 色彩計畫：例如「莫蘭迪色系」、「高對比黑白搭配鮮豔的紅色」、「溫暖的大地色系 (Earthy tones)」。
  • 📝 參考句型範例：「復古拍立得風格，帶有溫暖的黃色調與顆粒感，自然柔和的光線，懷舊且放鬆的氛圍 --ar 16:9」
  • 💡 提示：介面提供了「儲存設定/匯出/匯入」的功能，您可以將手感最好的咒語存下來，或與其他人交換風格檔。
⏱️ 演講時間預估儀表板： 當簡報結構生成完畢後，結果區塊上方會立刻顯示這份簡報的「總頁數」、「純中文字數」以及「預估演講時間」，幫助您精準掌控上台節奏！您在介面上修改大綱時，時間也會即時重算。
